Ce dépôt contient mes solutions pour les exercices de programmation Python (Module Back-end).

## 📁 Contenu du dépôt
* **Exercice 1** : Gestion des tranches d'âge avec conditions `if/elif/else`.
* **Exercice 2** : Gestionnaire de contacts interactif avec boucle `while` et listes.
* **Exercice 4** : Calculatrice sécurisée (gestion de la division par zéro).

## 🚀 Comment lancer les exercices
Assurez-vous d'avoir Python installé, puis lancez chaque fichier via le terminal :
`python3 nom_du_fichier.py`
